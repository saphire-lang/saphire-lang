*Im mading his prefix and only have Saf.fetch and Saf.compress*

# Install
```sh
git clone https://github.com/saphire-lang/saphire-lang.git ~/saf
```
```sh
cd ~/saf
```
```sh
chmod +x build
```
```
./build
```
